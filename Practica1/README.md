# PACMAN - Práctica 1 IPC1

## Descripción
Juego de PACMAN en consola desarrollado en Java como parte de la práctica 1 del curso Introducción a la Programación y Computación 1.

## Requisitos del Sistema
- Java JDK 8 o superior
- NetBeans IDE (opcional)

## Instrucciones de Ejecución

### Opción 1: Usando NetBeans
1. Abrir NetBeans
2. File → Open Project → Seleccionar carpeta "Practica1"
3. Ejecutar con F6 o click derecho en Pacman.java → Run File

### Opción 2: Usando línea de comandos
```bash
cd Practica1/src
javac pacman/*.java
java pacman.Pacman
